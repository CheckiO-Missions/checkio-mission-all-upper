Sprawdź, czy wszystkie znaki zapisane są wielkimi literami. Jeśli string jest pusty lub nie zawiera żadnej litery - funkcja powinna zwrócić True.

Wejście: String.

Wyjście: Boolean.

Przykład:

{% if interpreter.slug == "js-node" %} 
isAllUpper('ALL UPPER') == true
isAllUpper('all lower') == false
isAllUpper('mixed UPPER and lower') == false
isAllUpper('') == true
{% else %} 
is_all_upper('ALL UPPER') == True
is_all_upper('all lower') == False
is_all_upper('mixed UPPER and lower') == False
is_all_upper('') == True
{% endif %}

Warunek wstępny: Tekst zawiera litery a-z, A-Z, cyfry oraz spacje.
