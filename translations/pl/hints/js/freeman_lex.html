Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

w rzeczywistości ta misja jest bardzo prosta. Musi Pan utworzyć nowy ciąg znaków z początkowego, zamieniając go na wielkie litery (wszystkie znaki stają się wielkie) i porównać nowy ciąg z początkowym. Jeśli są one równe, oznacza to, że początkowy ciąg jest już górny.

Ok, od czego powinienem zacząć?

Proszę przeczytać o metodzie string.toUpperCase(). 

text.toUpperCase()

Co robić dalej?

Teraz proszę porównać początkowy ciąg znaków i nowy ciąg znaków z == comparison operator. 

text == text.toUpperCase()

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zawinąć kod w wyrażenie funkcyjne i zwrócić wynik porównania.

Oto całe rozwiązanie:

function isAllUpper(text: string): boolean {

    return text == text.toUpperCase();
}

Jasne, proszę pobrać!

W tym rozwiązaniu wyrażenie regularne jest używane do sprawdzania braku małych liter. 

function isAllUpper(text: string): boolean {

    return!/[a-z]/.test(text);
}

To rozwiązanie jest podobne do Pańskiego, ale wykorzystuje funkcję anonimową i operator trójargumentowy. 

var isAllUpper = (text: string): boolean =>
text === text.toUpperCase()? true: false
