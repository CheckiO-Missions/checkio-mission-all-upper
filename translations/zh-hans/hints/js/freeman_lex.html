这个任务其实很简单。你需要从初始值创建一个新的字符串，将其转换为大写字母（所有字符都变成大写字母），然后将新的字符串与初始值进行比较。如果它们相等，说明初始字符串已经是大写字母了。

好吧，我应该从什么开始呢？

请阅读string.toUpperCase() 方法。 

text.toUpperCase()

接下来该怎么做？

现在只需比较初始字符串和一个新的字符串，== comparison operator 。 

text == text.toUpperCase()

那么，我的最终解决方案应该是什么样子的呢？

你只需要把你的代码包装成函数表达式并返回比较的结果。

下面是整个解决方案：

function isAllUpper(text: string): boolean {

    return text == text.toUpperCase();
}
