指定された文字列がすべて大文字で構成されているかをチェックします。空文字列、もしくは英文字が 1 つも含まれていない場合、関数は True を返してください。

Input: str / String

Output: bool / Boolean

isAllUpper('ALL UPPER') == true
isAllUpper('all lower') == false
isAllUpper('mixed UPPER and lower') == false
isAllUpper('') == true
isAllUpper('444') == true
isAllUpper('55 55 5') == true
{% else %}

is_all_upper('55 55 5') == True
{% endif %}
