Ok, what should I start from?

Read about string.toUpperCase() method. 

text.toUpperCase()
What to do next?

Now just compare the initial string and a new one with == comparison operator. 

text == text.toUpperCase()
So, how my final solution should look like?

You just need to wrap your code into function expression and return the result of the comparison.

Here is the whole solution:

function isAllUpper(text: string): boolean {

    return text == text.toUpperCase();
}
